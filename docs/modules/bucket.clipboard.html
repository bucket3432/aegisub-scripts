Module bucket.clipboard

Drop-in replacement for Aegisub's clipboard module that uses external programs to interface with the system clipboard.

Requires the following external programs in PATH:

  • powershell (Windows)
  • pbpaste (macOS)
  • xclip or xsel (Linux and others)
